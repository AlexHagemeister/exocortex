# What this folder is

The user's media store. Obsidian's default attachment location points here. Agents may read files and embed-link them from wiki pages; they do not treat this folder as knowledge to ingest.

# Rules

- **Read and embed-link freely.** Cite paths when a wiki page needs the media.
- **Move, rename, or delete only when the user explicitly and specifically approves** — the files are theirs.
- Mind the sync-layer file-size limit (see `meta/DEPLOYMENT.md`); oversized binaries break sync.
- Zone row authority: CLAUDE.md. This README is the folder law that row points agents to read.
